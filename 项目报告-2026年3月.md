# 硫磺采购智能定价系统

## 项目概述

基于 Next.js 16.1.6 构建的硫磺采购智能定价系统，集成 AI 对话、知识图谱、价格预测等功能。

**GitHub 仓库**: https://github.com/weikangceng7-ai/TrustWise-Pricing
**在线演示**: https://sulfur-agent-web.vercel.app

---

## 2026年3月开发成果

### 一、核心功能

#### 1. Neo4j 知识图谱集成
- 企业关联网络可视化
- 供应商关系图谱
- 产业链上下游分析

#### 2. 企业差异化分析
- **一化** (`/enterprise/yihua`) - 企业详情与市场定位
- **鲁西** (`/enterprise/luxi`) - 竞争力分析
- **金正大** (`/enterprise/jinzhengda`) - 采购策略分析

#### 3. LSTM 价格预测
- 基于深度学习的价格趋势预测
- 历史数据训练与模型优化
- 可视化预测结果展示

#### 4. AI 智能对话
- 集成 DeepSeek (deepseek-v3-0324) 模型
- 支持多轮对话
- 硫磺市场问答系统

---

### 二、UI/UX 优化

- **Dashboard**: 深色渐变主题、动态光晕效果
- **玻璃态卡片**: 毛玻璃背景、阴影层次
- **交互动画**: 悬停效果、过渡动画
- **响应式设计**: 适配多端设备

---

### 三、技术栈

| 类别 | 技术 |
|------|------|
| 前端框架 | Next.js 16.1.6 + Turbopack |
| 数据库 | Neon PostgreSQL + Drizzle ORM |
| 图数据库 | Neo4j |
| AI 模型 | DeepSeek via qnaigc API |
| 部署平台 | Vercel |
| UI 组件 | shadcn/ui + Tailwind CSS |

---

### 四、项目结构

```
sulfur-agent-web/
├── src/
│   ├── app/              # 页面路由
│   ├── components/       # React 组件
│   ├── lib/              # 工具函数
│   └── hooks/            # 自定义 Hooks
├── public/               # 静态资源

```
